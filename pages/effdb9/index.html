pm2 笔记

# pm2 笔记

pm2 用来管理 nodejs 应用，非常的方便，一些常用的命令如下。

# 安装

npm install pm2 -g

#yarn global add pm2
1
2
3

# 使用

# 启动程序

pm2 start app.js --name app-name
1

# 列出运行程序

pm2 list
1

# 停止程序

pm2 stop app-name
1

# 重启程序

pm2 restart app-name
1

# 删除程序

pm2 delete app-name
